Slightly better links widget

Development
Sunday, October 3rd, 2010
Tweet

I use the default WordPress Links widget at the bottom of my theme (everything to the right of ‘recently’) to display links to other blogs, pictures, other sites, and what not. I believe the kids are calling that a ‘blogroll’.

The problem is, the default links widget orders by name alphabetically. I want links to appear with newest links first, older links last. The links widget doesn’t offer any options for sorting. So, I feexed it.

You can override any default WordPress widget by registering a new widget with the same name. The default widgets are declared in the file ‘default-widgets.php’ stored in wp-includes. You can view the code here: http://core.trac.wordpress.org/browser/trunk/wp-includes/default-widgets.php

To change the sort order I created a new widget class which simply extends the default links widget class, implementing only a constructor and the “widget” method. In my widget method I simply set the “orderby” argument for the “wp_list_bookmarks” to “id” (it defaults to “name”) and set “order” to “DESC”.

The default links widget is overridden by the class constructor, which registers the new widget under the same id as the old widget. Easy peasy.

If you want to do the same, add the following to your theme’s “functions.php” file.

/**
 * Override existing links widget, order links by ID descending so new links
 * appear at the top of the list
 */
class Abouthalf_Links extends WP_Widget_Links
{
    function Abouthalf_Links() {
        $widget_ops = array('description' => __( "Your blogroll" ));
        $this->WP_Widget('links', __('Links'), $widget_ops);
    }

    function widget( $args, $instance ) {
        extract($args, EXTR_SKIP);

        $show_description = isset($instance['description']) ? $instance['description'] : false;
        $show_name = isset($instance['name']) ? $instance['name'] : false;
        $show_rating = isset($instance['rating']) ? $instance['rating'] : false;
        $show_images = isset($instance['images']) ? $instance['images'] : true;
        $category = isset($instance['category']) ? $instance['category'] : false;

        if ( is_admin() && !$category ) {
            // Display All Links widget as such in the widgets screen
            echo $before_widget . $before_title. __('All Links') . $after_title . $after_widget;
            return;
        }

        $before_widget = preg_replace('/id="[^"]*"/','id="%id"', $before_widget);
        wp_list_bookmarks(apply_filters('widget_links_args', array(
            'title_before' => $before_title, 'title_after' => $after_title,
            'category_before' => $before_widget, 'category_after' => $after_widget,
            'show_images' => $show_images, 'show_description' => $show_description,
            'show_name' => $show_name, 'show_rating' => $show_rating,
            'category' => $category, 'class' => 'linkcat widget',
            'orderby' => 'id', 'order' => 'DESC'
        )));
    }
}

function AboutHalfLinksInit() {
    register_widget('Abouthalf_Links');
}
add_action('widgets_init', 'AboutHalfLinksInit');

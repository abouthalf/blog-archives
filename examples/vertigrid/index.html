A flexible grid with vertical alignment

Use inline-block instead of float

display: inline-block allows for vertical and horizontal alignment within your grid. Grid systems which use floats will always align to the top of current line.

Grid columns are displayed as inline-blocks and are contained within rows.

Rows and columns can be any element. This example uses ARTICLE elements for rows and DIV elements for columns.

Fix white space with font-size 0

Elements displayed as inline blocks will render white-space in between them just like the words in this sentence. This, of course, can break precise layouts. The solution is to set the containing element to have a font-size of 0px. Size-zero type has size-zero white space.

This requires a font-size reset for columns within the grid…

Reset font size for each column

If the base font size is 16px…

html
{
 font-size: 16px;
 font-size: 1em;
 line-height: 1.5em;
}
…and you're using relative fonts throughout…

h1
{
 font-size: 48px;
 font-size: 3rem;
 line-height: 1;
}

…style your grid rows to have a zero font size…

.row
{
 font-size: 0px;
}

…and reset each grid column to the base font size.

.s1, .s2, .s3, .s4
{
 display: inline-block;
 font-size: 16px;
 font-size: 1rem;
 margin-left: 24px;
}

Inline blocks are just responsive as floats

but are far more flexible.

Try resizing the browser window to a narrow width.[1]
Like floats, inline blocks will wrap if the containing element isn't wide enough to hold all the elements on a single line.
Mobile friendly designs are just as easy.

Offset columns by specifying an alignment on the row:
text-align: right;

Or by using margins:
margin-left: 810px;
/*(238 * 3) + (24 * 3)*/
